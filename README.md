# 📚 马原刷题系统

马克思主义基本原理课程刷题网站，**280 道题目**覆盖绪论至第四章全部内容。

## 题库

| 题型 | 数量 |
|---|---|
| 单选题 | 107 |
| 多选题 | 67 |
| 判断题 | 106 |
| **合计** | **280** |

## 功能

- 🎯 **刷题模式** — 按章节/题型筛选，顺序或随机出题
- 📋 **自主选题** — 搜索题目、按章节勾选，定制刷题范围
- 📖 **知识点查看** — 13 个核心知识点，左侧导航 + 搜索过滤
- ✅ 答对自动跳下一题，❌ 答错弹出解析弹窗

## 使用

直接用浏览器打开 `quiz.html` 即可，无需服务器。
